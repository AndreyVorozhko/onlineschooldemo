# Пример проекта онлайн-школы

Это пример кода на PHP, выполненного Андреем Ворожко.

Это онлайн-школа по типу "Геткурса", где нужно зайти в урок, изучить материалы урока, выполнить задание (если оно есть) и отправить на проверку куратору. Если это стоп-урок, то к следующуму уроку невозможно перейти пока текущий не одобрен куратором.

Проект написан без использования фреймворков на паттерне MVC

Использованы сторонние технологии (через composer): ORM RedBeanPHP, AdminLTE, шаблонизатор mustache и несколько других решений.

Это **пример кода**, а не готовый проект, тем не менее все работает (за незначительными исключениями).

Данные для входа:

Админка (http://`<yourdomain>`/admin): someproject@mail.ru:123456

Кабинет пользователя(http://`<yourdomain>`/user/area): anton@mail.ru:123456

Замечения по установке и настройке:

1. Дамп базы данных MySql находится в файле dump.sql
2. Настройки подключения к БД находятся в файле config/config_db.php (без настройки подключения к БД работать не будет)
3. Если появилась ошибка 404, включите дебаг в 3-й строке файла config/init.php:

```PHP
define("DEBUG", 1);
```
